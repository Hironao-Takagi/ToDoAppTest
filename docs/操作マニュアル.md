## 1. はじめに

ToDoリストアプリは、清掃業務のタスクを管理するためのWebアプリです。  
タスクの予定日・実績日を記録したり、定期的な清掃作業を繰り返しタスクとして登録することができます。

**アクセス方法**

ブラウザのアドレスバーにURLを入力してアクセスしてください。

---

## 2. 画面の見かた

### 2.1 全体レイアウト

```
┌─────────────────────────────────┐
│ ToDoリスト                       │  ← タイトル
│ タスクを管理してください           │
├─────────────────────────────────┤
│  [リスト]  [カレンダー]           │  ← 表示切替タブ
├─────────────────────────────────┤
│                                  │
│  （リストまたはカレンダーが表示）  │  ← メインエリア
│                                  │
└─────────────────────────────────┘
```

### 2.2 表示切替タブ

画面上部の **[リスト]** と **[カレンダー]** ボタンで表示を切り替えられます。

| タブ | 内容 |
|------|------|
| リスト | タスクを一覧で表示・追加・編集できます |
| カレンダー | 月間カレンダーでタスクの予定を確認できます |

## 3. タスクを追加する

### 3.1 手順

**1. タスク名を入力する**

画面上部の入力欄に、タスクの内容を入力します。  
例：「1F トイレ清掃」「エレベーター拭き掃除」

**2. 開始予定日・終了予定日を選択する**

日付選択欄をクリックして、カレンダーから日付を選びます。  
両方の日付の入力が必須です。

**3. 繰り返し設定をする（任意）**

繰り返し作業の場合は、繰り返しボタンから頻度を選択します。  
→ 詳しくは [9. 繰り返しタスクについて](#9-繰り返しタスクについて) を参照

**4. 「追加」ボタンを押す**

タスクが一覧に追加されます。

> **ヒント:** 入力欄でEnterキーを押しても追加できます。

### 3.2 入力項目

| 項目 | 必須 | 説明 |
|------|------|------|
| タスク名 | ○ | 作業内容を入力してください |
| 開始予定日 | ○ | 作業の開始予定日 |
| 終了予定日 | ○ | 作業の終了予定日 |
| 繰り返し | - | 定期作業の場合に選択（デフォルト：なし） |

### 3.3 エラーが表示された場合

| エラーメッセージ | 対処方法 |
|--------------|---------|
| 「タスク名を入力してください」 | タスク名の入力欄に文字を入力してください |
| 開始予定日が赤くなる | 開始予定日を選択してください |
| 終了予定日が赤くなる | 終了予定日を選択してください |

## 4. タスクを完了にする

### 4.1 手順

**1. 完了ボタンを押す**

タスク左端の **○ ボタン** を押します。実績日の入力パネルが表示されます。

**2. 実績日を入力する**

| 項目 | 説明 |
|------|------|
| 開始実績日 | 実際に作業を開始した日（本日の日付が自動入力されます） |
| 終了実績日 | 実際に作業を完了した日（本日の日付が自動入力されます） |

**3. 「完了する」ボタンを押す**

タスクが完了状態になり、テキストに打ち消し線が表示されます。

### 4.2 完了をキャンセルする

入力パネルの **「キャンセル」** ボタンを押すと、完了操作をキャンセルできます。

### 4.3 完了を取り消す（未完了に戻す）

完了済みタスクの **○ ボタン** を再度押すと、未完了の状態に戻ります。  
※ 実績日の記録はクリアされます。

> **注意:** 繰り返しタスクを完了すると、次の周期のタスクが自動的に作成されます。

## 5. タスクを編集する

タスクの日付情報を後から変更できます。

### 5.1 手順

**1. 展開ボタンを押す**

タスク右端の **∨ ボタン** を押すと、日付編集パネルが表示されます。

**2. 日付を変更する**

以下の4つの日付を変更できます。

| 項目 | 説明 |
|------|------|
| 開始予定日 | 作業の開始予定日 |
| 終了予定日 | 作業の終了予定日 |
| 開始実績 | 実際に開始した日 |
| 終了実績 | 実際に完了した日 |

**3. 「保存」ボタンを押す**

変更が保存され、編集パネルが閉じます。

**4. 編集をやめる場合**

再度 **∨ ボタン** を押すと、パネルが閉じます（変更は保存されません）。

## 6. タスクを削除する

### 6.1 個別削除

タスク右端の **× ボタン** を押すと、そのタスクが削除されます。

> **注意:** 削除したタスクは元に戻せません。

### 6.2 完了済みタスクの一括削除

画面下部の **「完了済みを削除」** ボタンを押すと、完了済みのタスクをまとめて削除できます。

> このボタンは完了済みタスクが1件以上ある場合のみ表示されます。

---

## 7. タスクを絞り込む

タスク一覧の上部にあるフィルタボタンで、表示するタスクを絞り込めます。

| ボタン | 表示されるタスク |
|--------|--------------|
| すべて | 完了・未完了すべてのタスク |
| 未完了 | 未完了のタスクのみ |
| 完了済み | 完了済みのタスクのみ |

画面下部には「**N件のタスクが残っています**」と未完了タスクの件数が表示されます。

## 8. カレンダーで確認する

### 8.1 カレンダーを開く

画面上部の **[カレンダー]** タブを押します。

### 8.2 月の切替

| ボタン | 操作 |
|--------|------|
| **<** | 前の月に移動 |
| **>** | 次の月に移動 |

### 8.3 タスクの色の意味

| 色 | 意味 |
|----|------|
| 紫色 | 未完了のタスク |
| グレー | 完了済みのタスク |
| 赤色 | 期限超過のタスク（終了予定日を過ぎた未完了タスク） |

### 8.4 日付を選択してタスクを確認する

カレンダー上の日付をクリックすると、その日のタスク一覧がカレンダーの下に表示されます。

> 1つのセルにタスクが3件以上ある場合、最大2件が表示され、残りは「+N」と表示されます。  
> 日付をクリックすると全件確認できます。

## 9. 繰り返しタスクについて

定期的な清掃作業など、同じタスクを繰り返す場合に設定します。

### 9.1 繰り返し種別

| 設定 | 動作 |
|------|------|
| なし | 指定した期間のみ表示（デフォルト） |
| 毎日 | 開始日以降、毎日表示 |
| 平日のみ | 開始日以降、月〜金のみ表示 |
| 毎週 | 選択した曜日に毎週表示 |
| 毎月 | 開始日と同じ日付に毎月表示（例：毎月3日） |
| 毎年 | 開始日と同じ月日に毎年表示（例：毎年4月3日） |

### 9.2 毎週の曜日選択

繰り返しを「**毎週**」に設定すると、曜日選択ボタンが表示されます。  
複数の曜日を選択することもできます。

例：月・水・金を選択 → 月曜・水曜・金曜に毎週表示

### 9.3 繰り返しタスクを完了する

繰り返しタスクを完了すると、**次の周期のタスクが自動的に作成されます**。

例：
- 毎週月曜のタスクを4月7日に完了 → 4月14日（次の月曜）のタスクが自動作成

> 繰り返しタスクを完全に終了したい場合は、新しく作成されたタスクを削除してください。

## 10. よくある質問

**Q. タスクを追加したのに画面に表示されない**  
A. フィルタが「未完了」または「完了済み」になっていないか確認してください。「すべて」を選択すると全タスクが表示されます。

**Q. ブラウザを閉じてもデータは保存されますか？**  
A. はい、データはブラウザのlocalStorageに自動保存されます。同じブラウザ・同じURLでアクセスすれば、再起動後もデータが残っています。

**Q. 別のパソコンからデータを見ることはできますか？**  
A. いいえ、データは使用しているブラウザにのみ保存されます。別のパソコンやブラウザからは参照できません。

**Q. 繰り返しタスクをカレンダーに表示したら日付ごとに何件も表示されます**  
A. 繰り返し設定により、該当する日付すべてに表示されます。完了処理を行うと、そのインスタンスは完了済みとなり、次の周期に新しいタスクが作成されます。

**Q. 誤って削除してしまいました。元に戻せますか？**  
A. 削除したタスクは元に戻せません。再度タスクを追加してください。

**Q. 期限超過（赤色）のタスクはどうすればいいですか？**  
A. 完了処理を行うか、日付編集パネルから終了予定日を更新してください。
