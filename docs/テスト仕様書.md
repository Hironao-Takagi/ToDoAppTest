## 1. テスト概要

### 1.1 目的

ToDoリストアプリケーションの各機能が仕様通りに動作することを確認する。

### 1.2 テスト範囲

| 対象機能 | テスト種別 |
|---------|-----------|
| タスク追加 | 機能テスト・バリデーションテスト |
| タスク完了 | 機能テスト |
| タスク編集 | 機能テスト |
| タスク削除 | 機能テスト |
| フィルタリング | 機能テスト |
| 繰り返し設定 | 機能テスト |
| カレンダー表示 | 機能テスト |
| データ永続化 | 機能テスト |
| セキュリティヘッダ | セキュリティテスト |

### 1.3 テスト判定基準

| 判定 | 記号 | 基準 |
|------|------|------|
| 合格 | ○ | 期待結果通りに動作する |
| 不合格 | × | 期待結果と異なる動作をする |
| 未実施 | - | テスト未実施 |

## 3. テストケース一覧

### 3.1 タスク追加（F-01）

| ID | テスト項目 | 手順 | 期待結果 |
|----|-----------|------|---------|
| T-01-01 | 正常追加 | ①タスク名を入力 ②開始予定日を選択 ③終了予定日を選択 ④「追加」ボタン押下 | タスクが一覧に追加される。入力フォームがリセットされる |
| T-01-02 | タスク名未入力 | ①タスク名を空のまま ②日付を入力 ③「追加」ボタン押下 | 「タスク名を入力してください」エラーが表示され、追加されない |
| T-01-03 | 開始予定日未入力 | ①タスク名を入力 ②開始予定日を空のまま ③終了予定日を入力 ④「追加」ボタン押下 | 開始予定日フィールドにエラー表示、追加されない |
| T-01-04 | 終了予定日未入力 | ①タスク名を入力 ②開始予定日を入力 ③終了予定日を空のまま ④「追加」ボタン押下 | 終了予定日フィールドにエラー表示、追加されない |
| T-01-05 | 全項目未入力 | ①何も入力せず「追加」ボタン押下 | タスク名・開始予定日・終了予定日の全てにエラー表示、追加されない |
| T-01-06 | Enterキーで追加 | ①全項目を入力 ②Enterキー押下 | タスクが追加される |
| T-01-07 | スペースのみのタスク名 | ①タスク名にスペースのみ入力 ②日付を入力 ③「追加」ボタン押下 | エラーが表示され追加されない（trim()後に空文字チェック） |

### 3.2 繰り返し設定（F-09）

| ID | テスト項目 | 手順 | 期待結果 |
|----|-----------|------|---------|
| T-02-01 | 繰り返しなし（デフォルト） | ①繰り返しを「なし」で登録 | 指定した予定期間のみタスクが表示される |
| T-02-02 | 毎日繰り返し | ①繰り返しを「毎日」で登録 | 開始日以降、カレンダーの全日にタスクが表示される |
| T-02-03 | 平日のみ繰り返し | ①繰り返しを「平日のみ」で登録 | 開始日以降の月〜金のみタスクが表示される。土日は表示されない |
| T-02-04 | 毎週繰り返し（曜日指定） | ①繰り返しを「毎週」で選択 ②曜日ボタンで「月」「水」を選択 ③登録 | 開始日以降の月曜・水曜のみタスクが表示される |
| T-02-05 | 毎月繰り返し | ①開始日を4月3日で「毎月」設定 ②登録 | 毎月3日にタスクが表示される |
| T-02-06 | 毎年繰り返し | ①開始日を4月3日で「毎年」設定 ②登録 | 毎年4月3日にタスクが表示される |
| T-02-07 | 繰り返し完了後の次インスタンス生成 | ①「毎週」設定のタスクを完了する（実績日入力） | 完了処理後、次の週の同曜日に新しいタスクが自動生成される |

### 3.3 タスク完了（F-03）

| ID | テスト項目 | 手順 | 期待結果 |
|----|-----------|------|---------|
| T-03-01 | 正常完了 | ①完了ボタン（○）押下 ②実績開始日・終了日を入力 ③「完了する」ボタン押下 | タスクが完了状態になる。テキストに打ち消し線が付く |
| T-03-02 | 実績日未入力で完了不可 | ①完了ボタン押下 ②実績日を空のまま「完了する」押下 | エラーが表示され完了されない |
| T-03-03 | 完了のキャンセル | ①完了ボタン押下 ②「キャンセル」ボタン押下 | 完了パネルが閉じ、タスクは未完了のまま |
| T-03-04 | 完了済みを未完了に戻す | ①完了済みタスクの完了ボタンを再押下 | タスクが未完了に戻り、実績日がクリアされる |
| T-03-05 | 実績日の初期値確認 | ①完了ボタン押下 | 実績開始日・終了日の初期値が本日の日付になっている |

### 3.4 タスク編集（F-04）

| ID | テスト項目 | 手順 | 期待結果 |
|----|-----------|------|---------|
| T-04-01 | 展開パネル表示 | ①タスク右端の展開ボタン（∨）押下 | 日付編集パネルが表示される（開始予定日・終了予定日・開始実績・終了実績） |
| T-04-02 | 日付の編集・保存 | ①展開ボタン押下 ②任意の日付を変更 ③「保存」ボタン押下 | 変更した日付が反映され、パネルが閉じる |
| T-04-03 | 展開パネルの折りたたみ | ①展開ボタン押下でパネル表示 ②再度展開ボタン押下 | パネルが折りたたまれる |
| T-04-04 | 完了パネルと展開パネルの排他制御 | ①完了ボタン押下（完了パネル表示中） ②展開ボタン押下 | 展開パネルが開き、完了パネルが閉じる |

### 3.5 タスク削除（F-05・F-06）

| ID | テスト項目 | 手順 | 期待結果 |
|----|-----------|------|---------|
| T-05-01 | 個別削除 | ①タスク右端の削除ボタン（×）押下 | タスクが一覧から削除される |
| T-05-02 | 完了済み一括削除 | ①完了済みタスクが1件以上存在する状態で「完了済みを削除」ボタン押下 | 完了済みタスクが全て削除される。未完了タスクは残る |
| T-05-03 | 完了済みなし時のボタン非表示 | ①完了済みタスクが0件の状態を確認 | 「完了済みを削除」ボタンが表示されない |

### 3.6 フィルタリング（F-07）

| ID | テスト項目 | 手順 | 期待結果 |
|----|-----------|------|---------|
| T-06-01 | 「すべて」フィルタ | ①「すべて」ボタン押下 | 完了・未完了すべてのタスクが表示される |
| T-06-02 | 「未完了」フィルタ | ①「未完了」ボタン押下 | 未完了タスクのみ表示される |
| T-06-03 | 「完了済み」フィルタ | ①「完了済み」ボタン押下 | 完了済みタスクのみ表示される |
| T-06-04 | 残件数表示 | ①タスクを複数登録し一部を完了 | フッターに「N件のタスクが残っています」と未完了件数が表示される |

### 3.7 カレンダー表示（F-08）

| ID | テスト項目 | 手順 | 期待結果 |
|----|-----------|------|---------|
| T-07-01 | カレンダー表示切替 | ①「カレンダー」タブ押下 | 月間カレンダーが表示される |
| T-07-02 | リスト表示への戻り | ①「リスト」タブ押下 | リストビューが表示される |
| T-07-03 | 前月・次月ナビ | ①「<」「>」ボタン押下 | 表示月が切り替わる。年をまたぐ場合も正常に動作する |
| T-07-04 | 日付クリックでタスク表示 | ①タスクが存在する日付をクリック | カレンダー下部にその日のタスク一覧が表示される |
| T-07-05 | 期限超過タスクの色表示 | ①終了予定日が過去のタスク（繰り返しなし・未完了）を登録 | カレンダー上で赤色表示される |
| T-07-06 | 完了済みタスクの色表示 | ①完了済みタスクが存在する日を確認 | グレー表示される |
| T-07-07 | 3件以上のタスク省略表示 | ①同じ日に3件以上タスクを登録 | カレンダーのセル内に最大2件表示され、超過分は「+N」と表示される |

### 3.8 データ永続化

| T-08-02 | ブラウザ再起動後のデータ保持 | ①タスクを追加 ②ブラウザを閉じて再起動 ③アプリにアクセス | タスクが保持されている |
